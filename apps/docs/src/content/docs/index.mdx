## Install

The runtime cone (`@anvilkit/core`, `ir`, `schema`, `validator`, `utils`,
`ui`) and every `@anvilkit/plugin-*` ship as their own npm packages,
**versioned independently** via Changesets — there is no lockstep release
train and no umbrella bundle. The line is currently `0.1.x`. Add only what
you need:

```bash
pnpm add @anvilkit/core @anvilkit/ir \
         @anvilkit/schema @anvilkit/validator \
         @anvilkit/plugin-ai-copilot \
         @anvilkit/plugin-asset-manager \
         @anvilkit/plugin-export-html \
         @anvilkit/plugin-export-react \
         @anvilkit/plugin-version-history
```

Realtime collaboration ships on a pre-release (`next`) dist-tag — read the
[Collaboration guide](/guides/collaboration/) before installing:

```bash
pnpm add @anvilkit/plugin-collab-yjs@next @anvilkit/collab-ui@next yjs y-protocols
```

The additive `PageIRNode.meta` authoring metadata is documented in the
[Page IR meta](/migration/page-ir-meta/) migration note.
