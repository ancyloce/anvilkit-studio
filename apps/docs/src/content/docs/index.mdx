## What's new in v1.1.0

The runtime cone (`@anvilkit/core`, `ir`, `schema`, `validator`, `utils`,
`ui`) and the five GA plugins (`plugin-ai-copilot`, `plugin-asset-manager`,
`plugin-export-html`, `plugin-export-react`, `plugin-version-history`) all
bump to `1.1.0` in lockstep. The release is **purely additive** over `1.0.0`;
the [LTS commitment](https://github.com/ancyloce/anvilkit-studio/blob/main/docs/policies/lts.md)
carries forward unchanged.

```bash
pnpm add @anvilkit/core@^1.1.0 @anvilkit/ir@^1.1.0 \
         @anvilkit/schema@^1.1.0 @anvilkit/validator@^1.1.0 \
         @anvilkit/plugin-ai-copilot@^1.1.0 \
         @anvilkit/plugin-asset-manager@^1.1.0 \
         @anvilkit/plugin-export-html@^1.1.0 \
         @anvilkit/plugin-export-react@^1.1.0 \
         @anvilkit/plugin-version-history@^1.1.0
```

Realtime collaboration ships separately on the alpha channel — see the
[announcement](https://github.com/ancyloce/anvilkit-studio/blob/main/docs/announcements/2027-12-v1-1-ga.md)
§ "Alpha caveats" before installing:

```bash
pnpm add @anvilkit/plugin-collab-yjs@alpha yjs y-protocols
```

Read the [migration guide](https://github.com/ancyloce/anvilkit-studio/blob/main/docs/migration/1.0-to-1.1.md)
for the full additive surface (section-level AI editing, `PageIRNode.meta`,
`anvilkit add`, `SnapshotAdapter` v2, `@anvilkit/ui/presence`).

## Component packages

Each component ships as its own `@anvilkit/*` npm package, versioned independently
via Changesets. Install only what you need — there is no umbrella bundle.
