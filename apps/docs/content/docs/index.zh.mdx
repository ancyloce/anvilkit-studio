AnvilKit 是一个可独立发布的 **Puck 原生 React
组件包**、运行时和插件生态系统的 monorepo。组件在 `@anvilkit/*`
命名空间下单独发布，并接入
[Puck](https://puckeditor.com/) —— 一个无头页面构建器。

<Callout title="迁移预览">
	本站点是文档的 Fumadocs + TanStack Start 重建版本。在切换之前，
	它与正在运行的 Astro/Starlight 站点并行运行。
</Callout>

## 这里有什么

- **组件** —— 只安装你需要的。
- **指南** —— 创作、插件、导出、协作。
- **API 参考** —— 由 TypeDoc 生成。
- **Playground** —— 一个具有完整插件功能对等的交互式 Puck 编辑器。
