AnvilKit は、独立して公開可能な **Puck ネイティブの React
コンポーネントパッケージ**、ランタイム、プラグインエコシステムの monorepo です。コンポーネントは `@anvilkit/*`
名前空間の下で個別に出荷され、
[Puck](https://puckeditor.com/) —— ヘッドレスなページビルダー —— に接続されます。

<Callout title="移行プレビュー">
	このサイトはドキュメントの Fumadocs + TanStack Start による再構築版です。切り替えが完了するまで、
	稼働中の Astro/Starlight サイトと並行して動作します。
</Callout>

## ここにあるもの

- **コンポーネント** —— 必要なものだけをインストールできます。
- **ガイド** —— オーサリング、プラグイン、エクスポート、コラボレーション。
- **API リファレンス** —— TypeDoc から生成。
- **Playground** —— 完全なプラグイン互換性を備えたインタラクティブな Puck エディター。
