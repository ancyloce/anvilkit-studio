AnvilKit is a monorepo of independently publishable **Puck-native React
component packages**, a runtime, and a plugin ecosystem. Components ship
separately under the `@anvilkit/*` namespace and plug into
[Puck](https://puckeditor.com/) — a headless page builder.

<Callout title="Migration preview">
	This site is the Fumadocs + TanStack Start rebuild of the docs. It runs
	alongside the live Astro/Starlight site until cutover.
</Callout>

## What's here

- **Components** — install only what you need.
- **Guides** — authoring, plugins, export, collaboration.
- **API reference** — generated from TypeDoc.
- **Playground** — an interactive Puck editor with full plugin parity.
