AnvilKit은 독립적으로 게시할 수 있는 **Puck 네이티브 React
컴포넌트 패키지**, 런타임 및 플러그인 생태계의 monorepo입니다. 컴포넌트는 `@anvilkit/*`
네임스페이스 아래에서 개별적으로 제공되며
[Puck](https://puckeditor.com/) —— 헤드리스 페이지 빌더 —— 에 연결됩니다.

<Callout title="마이그레이션 미리보기">
	이 사이트는 문서의 Fumadocs + TanStack Start 재구축 버전입니다. 전환이 완료될 때까지
	운영 중인 Astro/Starlight 사이트와 병행하여 실행됩니다.
</Callout>

## 여기에 있는 것

- **컴포넌트** —— 필요한 것만 설치하세요.
- **가이드** —— 작성, 플러그인, 내보내기, 협업.
- **API 참조** —— TypeDoc에서 생성됩니다.
- **Playground** —— 완전한 플러그인 호환성을 갖춘 인터랙티브 Puck 에디터.
