An enriched, persistable, replayable record of one applied command (FR-070).
Additive alongside [CanvasChange](../type-aliases/CanvasChange): `change` carries the existing
best-effort content diff, `command` carries the original command so a
sequence of records can be replayed deterministically via [replayChanges](../functions/replayChanges).
