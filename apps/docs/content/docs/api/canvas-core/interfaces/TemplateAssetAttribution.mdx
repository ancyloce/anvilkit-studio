Attribution metadata for a single asset a template references (FR-082,
canvas-m6-003) — who created/owns it, and the credit string required when
using it. Keyed by asset id in [CanvasTemplateDefinition.assetAttributions](CanvasTemplateDefinition.md#assetattributions);
distinct from [TemplateSourceMeta](TemplateSourceMeta.md), which describes the TEMPLATE's own
provenance, not a referenced asset's.
