```ts
function commandToChangeRecord(
   cmd, 
   ir,
```

Defined in: [commands/change-events.ts:160](https://github.com/ancyloce/anvilkit-studio/blob/main/src/commands/change-events.ts#L160)

Enrich a single command into a full [CanvasChangeRecord](../interfaces/CanvasChangeRecord), or `null`
for a `batch` (mirrors `commandToChange`; `applyCommands` maps sub-commands
individually). `ir` must be the pre-mutation IR so page/node lookups for
commands that omit `pageId` can resolve correctly.
