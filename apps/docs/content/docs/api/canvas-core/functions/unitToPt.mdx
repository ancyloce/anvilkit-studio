Defined in: [serialize/pdf.ts:44](https://github.com/ancyloce/anvilkit-studio/blob/main/src/serialize/pdf.ts#L44)

Convert a page dimension to PDF points. `in`/`mm` are physical; `px` is scaled
through `dpi` so a screen-pixel page maps to a sensible physical size. Mirrors
the unit semantics of `unitToPx` in `./svg.ts` (which shares [DEFAULT\_DPI](../variables/DEFAULT_DPI)).
