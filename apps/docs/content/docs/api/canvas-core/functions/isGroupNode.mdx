Defined in: [ir/walkers.ts:57](https://github.com/ancyloce/anvilkit-studio/blob/main/src/ir/walkers.ts#L57)

True only for a literal `group`. Prefer [isContainerNode](isContainerNode.md) when the
question is "does this node have children"; use this one only when the
behaviour is specific to groups (e.g. ungrouping, which a frame does not
support).
