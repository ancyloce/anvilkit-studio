### startGeneration()

```ts
startGeneration(prompt): void;
```

Defined in: [runtime/core/src/state/slices/ai-store.ts:115](https://github.com/ancyloce/anvilkit-studio/blob/main/packages/runtime/core/src/state/slices/ai-store.ts#L115)

Mark a new generation as in flight. Flips `isGenerating` to
`true`, stashes the prompt in `lastPrompt`, clears any prior
`lastError`, and appends a new [AiHistoryEntry](AiHistoryEntry.md) to
`history` with the current timestamp.

Always pair with a call to [finishGeneration](#finishgeneration) in a
`finally` block, or the store will report a phantom in-flight
state.

| `prompt` | `string` |
