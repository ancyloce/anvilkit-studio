Immutable snapshot of every sidebar contribution collected during a
single [compilePlugins](../functions/compilePlugins) pass, exposed on
[StudioRuntime.sidebar](StudioRuntime.md#sidebar). Map-keyed surfaces are last-write-wins
by id; the single-occupancy panels/source are last-write-wins by
registration order.
