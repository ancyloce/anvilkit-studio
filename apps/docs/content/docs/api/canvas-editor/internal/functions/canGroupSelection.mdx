```ts
function canGroupSelection(ir, selectedIds): boolean;
```

Defined in: [packages/capabilities/canvas/editor/src/selection/group-actions.ts:22](https://github.com/ancyloce/anvilkit-studio/blob/main/src/selection/group-actions.ts#L22)

True when the current selection can be grouped: at least two nodes, all
present on the active page and sharing the same immediate parent group.
Cross-parent grouping is intentionally unsupported (see `node.group`).
