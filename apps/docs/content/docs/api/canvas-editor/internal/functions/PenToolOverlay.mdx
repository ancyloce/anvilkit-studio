Keyboard finalizer for the pen tool (I3-2). Renders nothing; while the `path`
tool is active it listens globally for Enter (finalize an open path) and
Escape (cancel). Commit/cancel run through [commitPenPath](commitPenPath.md)/[cancelPenPath](cancelPenPath.md) — the same path the close-on-first-anchor click uses.

## Returns

`null`
