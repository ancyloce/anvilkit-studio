Interactive crop editor (I3-2). Renders an HTML overlay (like
[TextEditorOverlay](TextEditorOverlay.md)) over the cropping image with draggable corner
handles + a movable body; Enter / ✓ commits one `node.update`, Escape / ✕
cancels. Crop math runs through the pure [computeCropDrag](computeCropDrag.md); only the
commit touches `historyStore` (MVP-7).

`Element` \| `null`
