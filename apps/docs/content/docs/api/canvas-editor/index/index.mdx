## Classes

| Class | Description |
| ------ | ------ |
| [CanvasErrorBoundary](classes/CanvasErrorBoundary.md) | Error boundary for the canvas editor (W10). Without it, any render-time throw — a malformed IR, a missing asset, or a bug in a host-supplied panel — unwinds the whole React tree and the user loses the entire editor. Wrapping the panel registry and the stage subtree contains a failure to that region: the rest of the editor keeps working and the user gets a recover affordance instead of a blank screen. Exported so hosts can wrap their own custom panels too. |
