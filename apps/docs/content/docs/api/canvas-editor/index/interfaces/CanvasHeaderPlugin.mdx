A pluggable action mounted in the [WorkspaceHeader](../functions/WorkspaceHeader)'s right cluster
(between the collaborator avatars and the host `shareSlot`). `render` runs
*inside* the `<CanvasStudio>` provider, so the returned node may call
[useCanvasStudio](../functions/useCanvasStudio). `id` keys the node and dedupes registrations.
