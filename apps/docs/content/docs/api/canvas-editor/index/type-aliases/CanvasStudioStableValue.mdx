The stable half of [CanvasStudioContextValue](../interfaces/CanvasStudioContextValue) (W16): the store handles
and host callbacks, with NO per-commit live state (`ir`/`activePageId`/
`stage`). `<CanvasStudio>` memoizes this so its identity never changes after
mount, letting components that only need stores subscribe via
[useCanvasStores](../functions/useCanvasStores) and skip the re-render that fires on every edit for
consumers of the full [CanvasStudioContext](../variables/CanvasStudioContext).
