## Interfaces

| Interface | Description |
| ------ | ------ |
| [AiComponentSchema](interfaces/AiComponentSchema.md) | Description of a single component in a shape an LLM can reason about — name, purpose, props, and an optional concrete example. |
| [AiFieldSchema](interfaces/AiFieldSchema.md) | Description of a single prop on a component, in a shape an LLM can reason about. |
| [AiGenerationContext](interfaces/AiGenerationContext.md) | The full input to a single LLM generation call. |
| [AiSectionContext](interfaces/AiSectionContext.md) | The LLM-shaped view of a section-level regeneration request. |
| [AiSectionPatch](interfaces/AiSectionPatch.md) | The LLM's response to a section-level regeneration request. |
| [AiSectionSelection](interfaces/AiSectionSelection.md) | The editor's description of which nodes the author selected before asking for a section-level regeneration. |
| [AiValidationIssue](interfaces/AiValidationIssue.md) | A single issue produced by validating an LLM's response against the expected component/field shape. |
| [AiValidationResult](interfaces/AiValidationResult.md) | The outcome of validating an LLM response. |
| [AssetResolution](interfaces/AssetResolution.md) | The outcome of resolving an indirect asset reference (e.g. `asset://…`) to a concrete, directly-usable location. |
| [ConfigToAiSectionContextOptions](interfaces/ConfigToAiSectionContextOptions.md) | Optional environment hints for `configToAiSectionContext()`. |
| [ExportFormatDefinition](interfaces/ExportFormatDefinition.md) | The descriptor every exporter plugin contributes to Studio. |
| [ExportFormatRunContext](interfaces/ExportFormatRunContext.md) | Per-run context handed to [ExportFormatDefinition.run](interfaces/ExportFormatDefinition.md#run) for formats that opt into asset-URL rewriting (review finding TS-c). |
| [ExportResult](interfaces/ExportResult.md) | The return value of a successful [ExportFormatDefinition.run](interfaces/ExportFormatDefinition.md#run) call. |
| [ExportWarning](interfaces/ExportWarning.md) | A single diagnostic emitted by an exporter. |
| [PageIR](interfaces/PageIR.md) | The root IR document — the single argument passed to every [ExportFormatDefinition.run](interfaces/ExportFormatDefinition.md#run) call. |
| [PageIRAsset](interfaces/PageIRAsset.md) | An out-of-band resource referenced by the page — images, videos, fonts, stylesheets, inline scripts, etc. |
| [PageIRMetadata](interfaces/PageIRMetadata.md) | Page-level metadata — title, description, and timestamps the exporter may surface in `<head>`, a feed, or a manifest. |
| [PageIRNode](interfaces/PageIRNode.md) | A single node in the page IR tree. |
| [PageIRNodeMeta](interfaces/PageIRNodeMeta.md) | Authoring-time metadata attached to a [PageIRNode](interfaces/PageIRNode.md). |
| [StudioPage](interfaces/StudioPage.md) | A single page entry shown in the `layer/pages` sub-panel. |
| [StudioPageCreateInput](interfaces/StudioPageCreateInput.md) | Input shape for [StudioPagesSource.onCreate](interfaces/StudioPagesSource.md#oncreate). |
| [StudioPageRenameInput](interfaces/StudioPageRenameInput.md) | Input shape for [StudioPagesSource.onRename](interfaces/StudioPagesSource.md#onrename). |
| [StudioPageReorderInput](interfaces/StudioPageReorderInput.md) | Input shape for [StudioPagesSource.onReorder](interfaces/StudioPagesSource.md#onreorder). |
| [StudioPageSeo](interfaces/StudioPageSeo.md) | Structured SEO metadata for a page. Every field is optional so the host can surface only what it persists; the settings dialog shows the full set when the source implements [StudioPagesSource.onUpdateSettings](interfaces/StudioPagesSource.md#onupdatesettings). |
| [StudioPageSettingsInput](interfaces/StudioPageSettingsInput.md) | Input shape for [StudioPagesSource.onUpdateSettings](interfaces/StudioPagesSource.md#onupdatesettings). |
| [StudioPagesSource](interfaces/StudioPagesSource.md) | Pages source contract. The host application supplies one of these via the `<Studio>` `pages` prop. When omitted, the `layer/pages` sub-panel renders the empty state keyed `studio.module.layer.pages.empty`. |

## Type Aliases

| Type Alias | Description |
| ------ | ------ |
| [AiFieldType](type-aliases/AiFieldType.md) | The finite set of field types an AI schema can describe. |
| [AiThemeHint](type-aliases/AiThemeHint.md) | Theme hint forwarded into AI prompts so the model can pick appropriate colors, imagery, and tone (review finding TS-4). Named once here rather than inlining `"light" | "dark"` at each AI context site. (Kept local to the type-only `ai.ts` layer rather than reusing the react-layer `ThemeResolved`, which would cross the engine boundary.) |
| [ExportOptions](type-aliases/ExportOptions.md) | Generic bag of format-specific options passed to [ExportFormatDefinition.run](interfaces/ExportFormatDefinition.md#run). |
| [ExportWarningLevel](type-aliases/ExportWarningLevel.md) | Severity of a warning produced during an export run. |
| [IRAssetResolver](type-aliases/IRAssetResolver.md) | Rewrites an asset reference to its resolved location, or returns `null` when this resolver does not handle the reference. |
