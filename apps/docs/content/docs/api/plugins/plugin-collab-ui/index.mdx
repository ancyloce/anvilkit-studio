| [useCollabPeerIdentities](functions/useCollabPeerIdentities.md) | Roster-only peer identities (id/displayName/color). The returned array reference only changes when the *set* of identities changes — not on cursor / selection frames (F5) — so identity-only consumers like `<PeerAvatarStack>` skip cursor-churn re-renders.
