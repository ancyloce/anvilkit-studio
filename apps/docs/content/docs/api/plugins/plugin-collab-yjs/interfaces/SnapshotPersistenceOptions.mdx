§4.2.2 — configuration bag for [CreateYjsAdapterOptions.snapshotPersistence](CreateYjsAdapterOptions.md#snapshotpersistence).
Bundles the [SnapshotPersistenceAdapter](SnapshotPersistenceAdapter.md) with an optional
reversible payload transform (the encryption-at-rest seam) and a fault
sink.
