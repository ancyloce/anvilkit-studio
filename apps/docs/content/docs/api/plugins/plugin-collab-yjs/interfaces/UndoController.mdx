Local undo/redo surface exposed by [YjsSnapshotAdapter](YjsSnapshotAdapter.md) when
[CreateYjsAdapterOptions.undo](CreateYjsAdapterOptions.md#undo) is provided. All methods are
always present; when undo was not enabled they are inert
(`canUndo`/`canRedo` return `false`, `undo`/`redo`/`clearUndo` are
no-ops, `onUndoStackChange` never fires).

## Extended by

- [`YjsSnapshotAdapter`](YjsSnapshotAdapter.md)
