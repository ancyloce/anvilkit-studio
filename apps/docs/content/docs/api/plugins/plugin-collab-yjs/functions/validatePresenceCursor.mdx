Validate an untrusted value into a PresenceCursor.

Requires finite numeric `x` and `y` coordinates; any other shape (or a
`NaN`/`Infinity` coordinate) returns `null`.

## Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `value` | `unknown` | Untrusted cursor payload. |

## Returns

`PresenceCursor` \| `null`

The `PresenceCursor`, or `null` if invalid.
