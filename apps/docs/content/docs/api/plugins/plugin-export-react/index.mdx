## Interfaces

| Interface | Description |
| ------ | ------ |
| [AssetPlan](interfaces/AssetPlan.md) | The asset plan returned by `collectReactAssets`. |
| [AssetRewrite](interfaces/AssetRewrite.md) | A rewrite entry: for an asset URL, which binding identifier the emitter should substitute when it sees that URL in a prop. |
| [EmitReactResult](interfaces/EmitReactResult.md) | Return shape of `emitReact(ir, options)`. |
| [ImportManifest](interfaces/ImportManifest.md) | Deduplicated, deterministic list of imports emitted by the emitter. Sorted by (kind, source, binding) so the output is byte-stable across runs. |
| [ImportRecord](interfaces/ImportRecord.md) | A single import emitted at the top of the React source file. |
| [ReactExportOptions](interfaces/ReactExportOptions.md) | Public options bag for `@anvilkit/plugin-export-react`. |
| [ResolvedReactExportOptions](interfaces/ResolvedReactExportOptions.md) | Fully-resolved options after defaults are applied. Every field is required so internal call sites can destructure without narrowing. |
| [SerializedProp](interfaces/SerializedProp.md) | Result of serializing a single prop value. Mirrors the emitter's warning contract: every non-serializable input surfaces an `ExportWarning { code: "NON_SERIALIZABLE_PROP" }` and falls back to a commented-out placeholder so the emitted JSX still parses. |

| Type Alias | Description |
