## Functions

| Function | Description |
| ------ | ------ |
| [applyDiff](functions/applyDiff.md) | - |
| [checkRestoreConflict](functions/checkRestoreConflict.md) | Pure optimistic-concurrency check for a rollback. Hashes `currentIR` and compares it to `expectedBaseHash`: a mismatch means the document changed underneath the planned restore (a remote collaborator or a local edit), so blindly applying the snapshot would clobber that work. |
| [createPermissionDeniedError](functions/createPermissionDeniedError.md) | Convenience constructor for an authorization failure. Host and remote adapters call this when the current actor is not permitted to perform a version-history `operation` (`"save"`, `"load"`, `"list"`, `"delete"`, or `"restore"`), yielding a [VersionHistoryError](classes/VersionHistoryError.md) whose [VersionHistoryError.code](classes/VersionHistoryError.md#code) is `"PERMISSION_DENIED"` so callers can discriminate it without string-matching the message. |
| [createVersionHistoryPlugin](functions/createVersionHistoryPlugin.md) | - |

| [inMemoryAdapter](functions/inMemoryAdapter.md) | Ephemeral, in-process snapshot store. Uses the shared delta-chain so a long history costs roughly one keyframe + N small diffs instead of N full `PageIR` copies. Suitable for tests, demos, and transient sessions. |
| [isIRDiff](functions/isIRDiff.md) | Structural guard for a persisted [IRDiff](type-aliases/IRDiff.md) — a (possibly empty) array of well-formed [IRDiffOp](type-aliases/IRDiffOp.md)s. An empty array (no changes) is valid. |
