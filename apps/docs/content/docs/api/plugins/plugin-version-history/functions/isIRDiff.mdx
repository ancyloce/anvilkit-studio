Structural guard for a persisted [IRDiff](../type-aliases/IRDiff) — a (possibly empty) array of
well-formed [IRDiffOp](../type-aliases/IRDiffOp)s. Apply it at the storage trust boundary before
casting an untrusted/parsed payload to `IRDiff` and feeding it to
[applyDiff](applyDiff.md): a malformed op is rejected here instead of throwing an
opaque `DiffApplyError` (or silently misapplying) deep inside replay. An
empty array (no changes) is valid.
