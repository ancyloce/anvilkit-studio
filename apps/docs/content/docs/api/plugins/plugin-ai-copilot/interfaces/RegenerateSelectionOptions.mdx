Per-call options for [AiCopilotPluginInstance.regenerateSelection](AiCopilotPluginInstance.md#regenerateselection).

Forwards onto [ConfigToAiSectionContextOptions](https://github.com/ancyloce/anvilkit-studio/blob/main/packages/foundation/contracts/src/ai-section.ts) so callers can
thread theme / locale / `allowResize` hints into the context the LLM
sees without reaching for a separate API.
