Failure codes raised by the async data-source / folder / external-provider
surfaces (PRD 0002 §3.6). Distinct from [AssetValidationError](../classes/AssetValidationError)
(upload validation) and [AssetResolutionError](../classes/AssetResolutionError) (resolver domain) — in
particular `ASSET_NOT_FOUND` belongs to the resolver, never here.
