Narrow an upload backend to a [ResumableUploadAdapter](../interfaces/ResumableUploadAdapter). The single-shot
[UploadAdapter](../type-aliases/UploadAdapter) is a function; the resumable adapter is an object with
the four lifecycle methods, so a `typeof === "object"` plus method check
distinguishes them unambiguously.
