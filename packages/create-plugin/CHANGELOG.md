# @anvilkit/create-plugin

## 0.1.1

### Major Changes

- First `1.0.0` publish to npm `latest`.

  `@anvilkit/create-plugin` powers `npm create @anvilkit/plugin <name>` and is
  the recommended scaffold for new first- and third-party Puck plugins. This
  cut moves it from the internal `0.1.0` workspace path to a public release
  on the `latest` dist-tag, aligned with the `0.1.1` platform patch.
